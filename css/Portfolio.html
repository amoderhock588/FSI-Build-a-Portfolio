(A totally factual conversation)

King: That's the spirit! So all you have to do to pass my class is to write a 10-word sentence on what you're favorite thing about programming is.

Alex: But I've learned so many things.

King: Just pick one, I don't care which. Here, I'll help you get started. My favorite thing about programming is… There! That's already seven words! Only three more!

Alex: F… a… v… o… Oh, barnacles.

King: What's wrong?

Alex: My track pad broke.

King: Give me that! Here's a mouse.

Alex: A mouse! One of the most permanent of all coding utensils. There.

King: Fantastic, let me see it.

Alex: No, wait! I changed my mind!

King: I'm sure whatever you've written is fine, just let me see.

Alex: No, Don't look! It's not ready.

King: It's so simple, only 10 words! My favorite thing about coding is blankity, blankity, Blank!

Alex: I can do this! I can do this!

King: My favorite thing about coding is…! My favorite thing about coding is…!

Alex: I can do this! I can do this! Is it hot in here, King? Why is it so hot in here? My hand is cramping, King! Make it stop!

King: You only need three… more… words! Okay, let me see what you've written.

Alex: It's not ready yet.

King: It's okay, Alex. Show the teacher what you've written.

Alex: No!

King: Give it to me!

Alex: No!

King: Let me see it! My favorite thing about coding is… Uh… Well, the rest doesn't matter! You pass! You pass!

Alex: King, I don't feel like I really did anything.

King: That's how coding is supposed to feel!
